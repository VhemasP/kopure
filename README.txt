KOPURE HTML Static Website

Isi folder ini sudah berupa website HTML biasa, bukan React/Vite.
File utama:
- index.html
- style.css
- script.js
- assets/

Cara buka lokal:
1. Buka folder ini.
2. Double click index.html.

Cara hosting paling mudah:
1. Upload seluruh isi folder ini ke Netlify Drop.
2. Pastikan index.html berada di root folder, bukan di dalam subfolder tambahan.

Catatan edit penting:
1. Foto anggota masih placeholder.
   Ganti file berikut dengan foto asli jika sudah siap:
   - assets/team-rayhan.jpg
   - assets/team-renza.jpg
   - assets/team-vhemas.jpg
   Gunakan nama file yang sama agar tidak perlu mengubah kode.

2. Kontak WhatsApp dan email masih placeholder.
   Edit bagian kontak di index.html:
   - WhatsApp: Nomor WhatsApp Brand
   - Email: Email Brand
   - href tombol Pesan Sekarang jika ingin diarahkan ke WhatsApp.

3. Logo dan flyer asli sudah dimasukkan:
   - assets/logo-kopure.jpeg
   - assets/logo-icon.png
   - assets/flyer-kopure-1.png
   - assets/flyer-kopure-2.png

4. Jangan masukkan NIM, nomor HP pribadi, alamat rumah, scan KTP, atau KTM ke website publik.
